    🤖 AI System Design
    
    
    5 نماذج AI متكاملة — Arabic NLP first
    
    
    Model 1AI Recommendation Engine
    Hybrid (Content-based + Collaborative Filtering + LLM reranking). Stack: Python FastAPI → Scikit-learn → GPT-4o reranking. Responds in user's selected language.
    Target accuracy: 82%+ user satisfaction rate
    
    
    Model 2AI Chatbot — Multilingual NLP
    RAG (Retrieval Augmented Generation) فوق قاعدة بيانات الكليات المصرية. LLM: GPT-4o. Vector DB: pgvector. Streaming via SSE. Auto-detects Arabic / English / French and responds in same language.
    
    
    Model 3Salary Predictor
    Gradient Boosting Regressor (XGBoost) trained on Egyptian job market data. Features: college_type, specialization, governorate, years_exp. Output formatted in user's locale (Arabic numerals for AR).
    
    
    Model 4Alternative Recommendations
    Rule-based + ML: similarity score بين الكليات بناءً على subject areas، career outcomes، fee ranges.
    
    
    Model 5Behavior Tracking + Lang Router
    Real-time event tracking → Redis queue → async processing → تحديث user preference vector. lang_router.py routes Arabic prompts to ar-optimized system prompt for best NLP quality.
